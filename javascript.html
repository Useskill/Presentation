자바스크립트의 로고 이렇게 생겼다

웹사이트는 HTML,CSS,Javascript 를 사용한다고 앞서 말했고 HTML,CSS 를 보안하려고 나온것이 Javascript 다.
그래서 자바스크립트는 컴퓨터 언어로 쳐주지만 HTML 과 CSS는 언어로 쳐주지 않는다.
하지만 자바스크립트를 구사하기 위해서는 HTML 과 CSS가 필수이다.

그럼으로 한마디로 이 웹사이트는 HTML,CSS,Javascript 로 만들어진 웹사이트가 아니라 Javascript(자바스크립트) 웹사이트로 말한다.
